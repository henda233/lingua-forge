# LinguaForgeHelper 项目开发文档
基于deno+ts。
剪切板采用xsel。
## 一、工具功能
### 1.1 代码新增、修改需要的行号显示
指定文件路径和名称，读取文件内容在每行头部增加行号，最后输出修改文件内容，并复制在剪切板上。

### 1.2 多个文件合并
指定多个文件地址，读取文件后进行合并，将合并内容复制在剪切板上。

### 1.3 AI提示词快速选择[废弃不实现]
快速读取指定预设的提示词，复制在剪切板上。
预设的提示词以硬编码的方式写入代码中。

### 1.4 剪切板
为1.1到1.3功能提供与系统剪切板交互的功能。
采用xsel实现，通过调用xsel CLI实现将内容复制到剪切板中。
这里的剪切板指ctrl+V和ctrl+C操作。
具体实现为：
    创建一个剪切板操作接口；
    采用xsel去定义类实现这个接口。
这样就可以保证不同系统、架构都统一API，自行具体实现。

### 1.4 CLI参数
#### 1.4.1 功能1.1
-l <file_path>  :功能1.1，file_path为指定的文件路径和名称
#### 1.4.2 功能1.2
-m <file_path_1> <file_path_2> ... :功能1.2
#### 1.4.3 功能1.3 [废弃不实现]
-p <prompt_index>   :功能1.3,prompt_index为要读取的提示词序号。如果不输入prompt_index，则显示有哪些提示词。

## 二、工具信息
工具命名为LinguaForgeHelper，CLI命令简称lf-helper。
版本0.0.1

## 三、工具运行流程
用户shell输入lf-helper -l main.ts等参数使用工具。
